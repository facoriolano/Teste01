# ✨ Prova do Crisma 26

Aplicativo web estilo **Duolingo**, responsivo para celular, criado para auxiliar na preparação da **Turma de Crisma 26**.  
As questões são carregadas diretamente de um **Google Sheets (CSV)**, permitindo prática interativa e dinâmica.
